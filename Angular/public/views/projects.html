Network Monitor

Java, Swing

When I first thought about doing this project I imagined it would take a lot more work to accomplish the final goal. In reality it was easier but still taught me a few things. The main part of this project required a simple command to the command console to get what the current network information was. I was aware that this was possible within Java but had not done it previously. Once I got this working it was simple matter of putting it in a loop to always get the current value. At this point everything is going to the console within IntelliJ IDE. I decided I wanted to create a GUI interface for it in order to keep up the little bit of Swing skill I had. For whatever reason implementing the GUI was the most rewarding aspect since it was a little finicky with some things. In the end I accomplised my goal and picked up a few extra skills as well.

Highway System

Java, Swing

I had a lot of fun creating this project. I originally started out just making a car object and playing around with its properties to keep up my basic OO skills. From there I thought about having a road with multiple static cars, but decided to take it a step further and make a whole highway with multiple lanes of moving cars. The a challenging part was trying to pick the best container to hold the cars. I eventually went with a HashMap for the highway and a Vector for each lane. The HashMap provided me the ability to map each lane to a number which provided easy key/value linking. Vectors were an efficient solution to the ever changing car positions and allowed for indexes to be found and updated quickly. After this it was a matter of adding cars to the correct lane and making sure they moved through the lane as expected. The last bit of functionality I added was to give the cars different speeds and make it impossible to overlap a car moving at a slower speed. This seemed simple at first but there was the challenge of giving the car the ability to "see" in front of it to know if it had to slow down or not. Once this hurdle was overcome I slapped the output into a GUI interface and the project was "finished" for now.
